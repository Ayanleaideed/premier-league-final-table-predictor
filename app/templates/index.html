Premier League Predictor

Table, points, and storyline in one view

Drop in your latest season snapshot or upload a raw match log. The FastAPI service loads the Random Forest stored at src/Training/rf_model_2024_25.joblib, recreates the medallion gold features, and sends back projected points plus positions.

Required fields per club: team, wins, draws, losses, goalsFor, goalsAgainst, homeWins, awayWins, and isPromoted. Matches played and points are filled in when missing.

Workflow

  1. Edit the JSON payload so it reflects the table you want to score (20 rows for the full league).
  2. Hit Predict Table or upload a CSV/XLS file if that’s easier.
  3. Review the response table and keep iterating. Nothing is stored server-side.

JSON playground

Upload standings or match logs

Column names are case-insensitive. The app recognizes team, wins, draws, losses, GF, GA, homeWins, awayWins, and isPromoted. Uploading match-level files with HomeTeam/AwayTeam/FTHG/FTAG is also supported.
